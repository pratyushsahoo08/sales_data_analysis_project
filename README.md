# sales_data_analysis_project
Overview

This project focuses on analyzing a company's yearly sales dataset to extract meaningful business insights.
The analysis includes data cleaning, SQL-based data querying, and dashboard visualization.

Tools Used:
Excel
SQL
Power BI

Data Cleaning (Excel):
The raw dataset was cleaned using Excel and Power Query before analysis.
The following steps were performed:
Removed duplicate records
Filled or handled null values
Detected and corrected incorrect data types
Corrected category name inconsistencies using Power Query
Verified calculated columns such as revenue

Data Analysis (SQL):
SQL queries were used to extract insights from the dataset.
Using SQL syntax and aggregation functions, the following metrics were calculated:
Total Revenue
Product-wise Revenue
Category-wise Revenue
Average Revenue per Sale
Total Quantity Sold
Quarterly Revenue Analysis

Data Visualization (Power BI):
Power BI was used to create interactive dashboards and visualizations to better understand sales performance and trends.
The dashboard includes:
Revenue by Category
Revenue by Product
Quarterly Revenue Distribution
Total Revenue and Quantity KPIs
Monthly Sales Trend

Key Insights :
Total Revenue: 759K
Highest Revenue Category: Electronics – 516K
Highest Revenue Product: Smartphone – 434K
Average Revenue per Sale: 2.06K
Total Quantity Sold: 5341 units
Highest Revenue Quarter: Q3 generated 26.04% of total sales

Business Observation:
Sales showed fluctuations in revenue during the first half of the year.
However, after July, sales became more stable with less variation, indicating a more consistent sales performance in the later months.

Conclusion:
This project demonstrates the complete data analysis workflow including data cleaning, SQL analysis, and Power BI visualization to generate meaningful business insights from raw sales data.
